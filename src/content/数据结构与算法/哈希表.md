# 哈希表专题笔记

## 0. 一句话理解

```text
把"过去见过的信息"存起来，让当前元素可以 O(1) 查询。
```

`unordered_map<int, int>` 可以表示：数字→下标、数字→出现次数、前缀和→出现次数、字符→上一次出现位置、字符串特征→一组字符串。

---

# 1. 六种核心模型

## 1.1 有没有出现过 → `unordered_set`

```cpp
unordered_set<int> seen;
for (int x : nums) {
    if (seen.count(x)) return true;
    seen.insert(x);
}
```

典型题：判断重复元素、最长连续序列、滑动窗口去重。

---

## 1.2 某个值对应什么信息 → `unordered_map<Key, Value>`

常见映射：数字→下标、数字→频率、字符→最近位置、前缀和→次数、排序后的字符串→异位词分组。

```cpp
unordered_map<int, int> pos;
pos[value] = index;
```

---

## 1.3 计数

```cpp
unordered_map<int, int> cnt;
for (int x : nums) cnt[x]++;
```

结果示例：`nums = [1,2,1,3,2,1]` → `{1:3, 2:2, 3:1}`。

典型题：Top K Frequent Elements、Find All Anagrams、Group Anagrams。

---

## 1.4 当前元素找过去的配对（Two Sum 型）

先查，再放。不能先放再查，否则会自己匹配自己。

```cpp
unordered_map<int, int> mp; // 数值 -> 下标
for (int i = 0; i < nums.size(); i++) {
    int need = target - nums[i];
    if (mp.count(need)) return {mp[need], i};
    mp[nums[i]] = i;
}
```

---

## 1.5 当前前缀和找过去的前缀和

核心公式：`子数组和 = 当前前缀和 - 过去某个前缀和`，变形得 `old_sum = current_sum - k`。

```cpp
unordered_map<int, int> cnt;
cnt[0] = 1;  // 数组开始前，前缀和为 0
int sum = 0, ans = 0;
for (int x : nums) {
    sum += x;
    ans += cnt[sum - k];
    cnt[sum]++;
}
```

# 2. 数据结构速查

## 2.1 四种容器对比

| 数据结构            | 内容      | 有序 | 底层  | 平均复杂度    | 场景     |
| --------------- | ------- | -- | --- | -------- | ------ |
| `unordered_map` | key-val | 否  | 哈希表 | O(1)     | 快速映射   |
| `unordered_set` | key     | 否  | 哈希表 | O(1)     | 快速判断存在 |
| `map`           | key-val | 是  | 红黑树 | O(log n) | 有序映射   |
| `set`           | key     | 是  | 红黑树 | O(log n) | 有序集合   |

默认用 `unordered_map` / `unordered_set`。需要有序、`lower_bound`、按 key 遍历时，才用 `map` / `set`。

## 2.2 数组当哈希表

key 范围小时，数组比哈希表更快：

```cpp
vector<int> cnt(26, 0);       // 小写字母
vector<int> cnt(128, 0);      // ASCII
vector<int> cnt(100001, 0);   // 较小非负整数

// 有负数时加偏移
cnt[x + offset]++;
```

# 3. 常用 API

## 3.1 unordered_map

```cpp
mp[key] = value;     // 插入/修改
mp[key]++;           // 计数（key 不存在时自动初始化为 0）
mp.count(key);       // 判断是否存在（推荐，不会创建 key）
mp[key];             // 访问（警告：key 不存在会自动创建！）
mp.erase(key);       // 删除
mp.size();
mp.clear();

// 遍历
for (const auto& [key, value] : mp) { ... }
```

**坑：** 判断是否存在要用 `mp.count(x)`，不要用 `if (mp[x])`——后者会把 key 插入 map。

## 3.2 unordered_set

```cpp
st.insert(x);
st.count(x);   // 判断是否存在
st.erase(x);
st.size();
st.clear();

unordered_set<int> st(nums.begin(), nums.end()); // 从数组构建，自动去重
```

## 3.3 map / set 额外 API

```cpp
auto it = st.lower_bound(x);  // 第一个 >= x 的元素
auto it = st.upper_bound(x);  // 第一个 > x 的元素
if (it != st.end()) cout << *it;
```

# 4. 六类题型详解

## 4.1 Two Sum 型

**模板：** 见 1.4 节。关键：先查 `target - nums[i]`，再放 `nums[i]`。

## 4.2 分组型（Group Anagrams）

给每个字符串提取"特征 key"，相同特征的放同一个桶。

```cpp
vector<vector<string>> groupAnagrams(vector<string>& strs) {
    unordered_map<string, vector<string>> groups;
    for (string s : strs) {
        string key = s;
        sort(key.begin(), key.end());  // 注意：sort 是原地修改，不返回值
        groups[key].push_back(s);      // push 原字符串 s，不是 key
    }
    vector<vector<string>> res;
    for (const auto& [key, group] : groups) res.push_back(group);//c++20结构化绑定语法
    return res;
}
```

---

## 4.3 最长连续序列（Longest Consecutive Sequence）

只从序列起点（`x-1` 不存在）开始扩展，保证每个元素最多被访问一次，时间复杂度 O(n)。

```cpp
int longestConsecutive(vector<int>& nums) {
    unordered_set<int> st(nums.begin(), nums.end());
    int ans = 0;
    for (int x : st) {           // 遍历 st 而非 nums，避免重复元素重复检查
        if (!st.count(x - 1)) {  // 只从起点开始
            int cur = x, len = 1;
            while (st.count(cur + 1)) { cur++; len++; }
            ans = max(ans, len);
        }
    }
    return ans;
}
```

## 4.4 前缀和 + 哈希表（Subarray Sum Equals K）

**触发关键词：** 连续子数组、子数组和、和为 k、统计个数、数组有负数。

三种变体：

```cpp
// 统计和为 K 的子数组个数
int subarraySum(vector<int>& nums, int k) {
    unordered_map<int, int> cnt;
    cnt[0] = 1;
    int sum = 0, ans = 0;
    for (int x : nums) {
        sum += x;
        ans += cnt[sum - k];
        cnt[sum]++;
    }
    return ans;
}

// 是否存在和为 K 的子数组
bool existsSubarraySumK(vector<int>& nums, int k) {
    unordered_set<int> seen;
    seen.insert(0);
    int sum = 0;
    for (int x : nums) {
        sum += x;
        if (seen.count(sum - k)) return true;
        seen.insert(sum);
    }
    return false;
}

// 最长和为 K 的子数组（存第一次出现位置，越早子数组越长）
int maxSubArrayLen(vector<int>& nums, int k) {
    unordered_map<int, int> first;
    first[0] = -1;
    int sum = 0, ans = 0;
    for (int i = 0; i < nums.size(); i++) {
        sum += nums[i];
        if (first.count(sum - k)) ans = max(ans, i - first[sum - k]);
        if (!first.count(sum)) first[sum] = i;
    }
    return ans;
}
```

## 4.5 滑动窗口 + 哈希表

**触发关键词：** 连续子串/子数组、最长/最短、不重复、包含、至多 K 个、字符频率、固定长度窗口。

```cpp
// 无重复字符最长子串（set 写法）
int lengthOfLongestSubstring(string s) {
    unordered_set<char> window;
    int left = 0, ans = 0;
    for (int right = 0; right < s.size(); right++) {
        while (window.count(s[right])) { window.erase(s[left++]); }
        window.insert(s[right]);
        ans = max(ans, right - left + 1);
    }
    return ans;
}

// 优化：记录上次出现位置（注意要用 max 防止 left 倒退）
int lengthOfLongestSubstring(string s) {
    unordered_map<char, int> last;
    int left = 0, ans = 0;
    for (int right = 0; right < s.size(); right++) {
        char c = s[right];
        if (last.count(c)) left = max(left, last[c] + 1);
        last[c] = right;
        ans = max(ans, right - left + 1);
    }
    return ans;
}

// 固定长度窗口（Find All Anagrams）
vector<int> findAnagrams(string s, string p) {
    vector<int> ans;
    if (s.size() < p.size()) return ans;
    vector<int> need(26, 0), window(26, 0);
    for (char c : p) need[c - 'a']++;
    int len = p.size();
    for (int right = 0; right < s.size(); right++) {
        window[s[right] - 'a']++;
        if (right >= len) window[s[right - len] - 'a']--;
        if (window == need) ans.push_back(right - len + 1);
    }
    return ans;
}
```

## 4.6 Top K 频率（Top K Frequent Elements）

```cpp
vector<int> topKFrequent(vector<int>& nums, int k) {
    unordered_map<int, int> cnt;
    for (int x : nums) cnt[x]++;

    vector<pair<int, int>> arr(cnt.begin(), cnt.end());
    sort(arr.begin(), arr.end(), [](auto& a, auto& b) {
        return a.second > b.second;
    });

    vector<int> res;
    for (int i = 0; i < k; i++) res.push_back(arr[i].first);
    return res;
}
```

# 5. 前缀和 vs 滑动窗口

| 对比点     | 前缀和 + 哈希表   | 滑动窗口 + 哈希表     |
| ------- | ----------- | -------------- |
| 核心对象    | 子数组和        | 窗口状态           |
| 哈希表存什么  | 过去的前缀和      | 当前窗口的元素频率      |
| 指针特点    | 一个指针扫描      | left/right 双指针 |
| 适合负数求和  | 是           | 否              |
| 触发关键词   | 子数组和、k、统计个数 | 子串、窗口、不重复、包含   |

**为什么有负数不能用滑动窗口：** 全正数时 right 右移 sum 单调增，有单调性可以控制收缩；有负数时单调性消失，窗口收缩逻辑无法稳定。

# 6. 常见坑点

**`mp[x]` 会自动创建 key：** 判断存在要用 `mp.count(x)`，不要用 `if (mp[x])`。

**`sort` 不返回容器：**
```cpp
// 错误
string key = sort(s.begin(), s.end());
// 正确
string key = s;
sort(key.begin(), key.end());
```

**前缀和初始化别忘：** 统计个数用 `cnt[0] = 1`，最长子数组用 `first[0] = -1`。

**滑动窗口删除时同步更新哈希表：** 移动 `left` 的同时，必须 `window[s[left]]--` 或 `window.erase(s[left])`。

**窗口收缩用 `while` 不用 `if`：** 移除一个元素后窗口可能仍不合法，需要继续收缩。

**遍历 set 而非 nums（最长连续序列）：** `nums` 可能有重复，重复元素会触发重复检查。

# 7. 字符频率匹配

只有小写字母时，数组比哈希表更快：

```cpp
bool isAnagram(string s, string t) {
    if (s.size() != t.size()) return false;
    vector<int> cnt(26, 0);
    for (char c : s) cnt[c - 'a']++;
    for (char c : t) cnt[c - 'a']--;
    for (int x : cnt) if (x != 0) return false;
    return true;
}
```
