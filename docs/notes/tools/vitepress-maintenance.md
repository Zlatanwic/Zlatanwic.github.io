# VitePress 站点维护清单

一个长期站点最怕的不是没有功能，而是越来越难维护。所以我倾向于保留一份简洁的维护清单。

## 内容层

- 首页入口是否仍然清晰
- 博客是否有最新文章可见
- 知识库索引页是否还有效
- 是否存在没有回链的孤立页面

## 结构层

- `nav` 是否仍然对应当前内容重心
- `sidebar` 是否覆盖了主要目录
- 新增页面有没有放到正确分类

## 视觉层

- 首页是否过度拥挤
- 各页面标题层级是否一致
- 重点按钮和入口是否仍然明显

## 发布层

- 构建是否通过
- 静态资源路径是否正常
- SEO 基础信息是否缺失

一份小清单最大的价值，不是看起来完整，而是每次迭代时真的会被用到。
