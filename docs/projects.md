# 项目

这里记录我正在做、已经做过，或者准备长期迭代的事情。

## 当前重点

### 个人数字花园

用 VitePress 维护一个同时兼顾主页、博客和知识库的站点，目标不是单纯“把网页做出来”，而是建立一个可以持续生长的内容系统。

关注点：

- 信息架构是否清晰
- 内容是否能互相链接
- 首页是否既能表达个人气质，又不过度装饰

### 写作系统

我在持续迭代一套更轻量的写作流程，让“想到一个主题”到“发布成一篇文章”之间的摩擦尽量小。

关注点：

- 草稿如何分层
- 什么时候应该写成博客，什么时候应该沉淀为知识卡片
- 如何避免知识库越来越乱

## 我偏爱的项目类型

- 能长期复利的个人基础设施
- 能把想法沉淀成系统的工具
- 能同时提升表达、组织和交付效率的项目
