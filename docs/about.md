# 关于我

你好，我是 李阔。

我是同济大学的学生，平时主要关注

## 我关注什么

我现在持续关注的方向主要有这些：

- 计算机系统与底层原理
- 编程语言与运行时
- 软件工程与 system design
- AI 工具、智能体和相关基础设施
- 知识管理、写作与个人工作流

这些主题看起来不完全一样，但对我来说，它们都和同一个问题有关：复杂系统是如何被构建、理解和维护的。

## 我比较在意的东西

我学技术时，通常会比较在意这些问题：

- 一个工具或框架解决了什么问题
- 它为什么这样设计
- 它的边界和代价是什么
- 如果把它放进更大的系统里，会产生什么影响

写代码也是一样。我会更关注模块边界、抽象是否清晰、设计是否能长期维护，而不只是先把功能拼出来。

## 我平时喜欢做的事

除了写代码本身，我也对这些事情比较感兴趣：

- 配环境和整理工具链
- 搭建自动化流程
- 研究构建、部署和工程化细节
- 把零散的经验整理成可复用的知识

这些事情不一定显眼，但往往决定了一个项目后面是否顺手、稳定和可持续。
